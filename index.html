1. Emphasis & Importance

Semantic tags help accessibility (screen readers) & indexing. Use semantic tags for meaning — visual tags for style only.

Tag Meaning Code (copy) & Rendered output
<em> Emphasized (semantic) 
<p>When speaking, <em>emphasis</em> changes tone.</p>

When speaking, emphasis changes tone.
<strong> Strong importance (semantic) 
<p>This is <strong>important</strong> — act now.</p>

This is important — act now.
<i> Alternate voice / foreign term (visual) 
<p>The phrase <i>per se</i> is Latin.</p>

The phrase per se is Latin.
<b> Stylistic bold (no semantic weight) 
<p>Remember this <b>keyword</b> in the UI.</p>

Remember this keyword in the UI.
<small> & <mark> Side notes & highlights 
<p>Terms apply. <small>Offer ends tonight.</small> Search shows <mark>highlighted</mark> matches.</p>

Terms apply. Offer ends tonight. Search shows highlighted matches.

Tip: If you want screen readers to emphasize, use <em> — if you only want bold text visually (no change to meaning) use <b>.

CSS Fundamentals — Expanded

Concise foundation: cascade, specificity, box model, layout, responsiveness, typography.

Cascade & Specificity

/* selector specificity example */
p { color: blue; }         /* element = low */
.myclass { color: green; } /* class = higher */
#myid { color: red; }      /* id = even higher */
/* inline styles beat all (except !important) */

Rendered text shows how different selectors override one another depending on specificity.

Inheritance

body { color: #333; font-family: Inter; }
section { /* section inherits color & font unless overridden */ }

Child elements inherit text color and font-family by default.

Box Model

.box { box-sizing: border-box; width:200px; padding:10px; border:2px solid #ddd; }
.box — width includes padding & border

Layout Basics

/* Flex example */

Positioning

position: static|relative|absolute|fixed|sticky;

Use position:sticky; top:0; to pin toolbars. Use absolute when removing from flow.

Responsive Tips

/* mobile-first */
.container{ width:100%; max-width:1200px; margin:0 auto; padding:12px; }

/* fluid text */
h1{ font-size:clamp(20px, 4vw, 36px); }

Use relative units (%, em, rem, vw) and clamp() for fluid typography. Add viewport meta (already present).

Typography & Spacing

h1,h2,h3 { line-height:1.4; }
p { line-height:1.6; margin-bottom:12px; }

Readable line-height (~1.5) and consistent vertical rhythm (spacing) improve UX.

Color & Contrast

/* ensure contrast >= 4.5:1 for body text */

Avoid pure black (#000) on pure white (#fff) for comfort; prefer near-black & off-white.
